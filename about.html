The Project

What is KidneyCare AI?

KidneyCare AI is a clinical decision-support tool designed to assist healthcare professionals and patients in identifying the early signs of Chronic Kidney Disease (CKD). Using a curated set of nine commonly measured biomarkers, our machine learning model classifies patients as CKD positive or CKD negative with a high degree of accuracy.

CKD affects approximately 10% of the global population and is a leading cause of kidney failure and cardiovascular disease. Early detection is critical — patients identified in stages 1–3 can often slow or halt disease progression through lifestyle changes and medication.

This tool was built as an academic and educational project using the UCI Chronic Kidney Disease dataset, which contains data from 400 patients collected over a two-month period.

Under the Hood

Technologies Used

A lean, modern stack chosen for reliability, speed, and healthcare-grade quality.

Flask

Python Web Framework

A lightweight WSGI web framework used to serve the ML model as a RESTful API. Flask's minimalist design makes it ideal for healthcare microservices and rapid prototyping.

  • REST API endpoint for model inference
  • CORS-enabled for frontend communication
  • Blueprint-based modular routing

Machine Learning

Predictive Modeling

The CKD classifier is trained using ensemble learning techniques — specifically a Random Forest classifier — to handle missing values and categorical features common in clinical datasets.

  • Random Forest Classifier
  • Cross-validation with stratified k-fold
  • Feature importance ranking
  • Handles class imbalance via SMOTE

Scikit-learn

ML Library

Python's industry-standard machine learning library provides all preprocessing pipelines, model training, evaluation metrics, and serialization tools used in this project.

  • Pipeline API for preprocessing + model
  • StandardScaler & LabelEncoder
  • GridSearchCV for hyperparameter tuning
  • Joblib model serialization
Training Data

UCI CKD Dataset

The model was trained on the UCI Machine Learning Repository – Chronic Kidney Disease dataset, collected at a hospital in Tamil Nadu, India over two months. The dataset includes 24 features across 400 patients.

For this web interface, we use the 9 most predictive features based on feature importance analysis, making the tool both accurate and user-friendly.
